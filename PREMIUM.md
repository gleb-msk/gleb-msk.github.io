# Премиум-версия лендинга (ultra premium)

Ультра-премиум версия: визуальная иерархия, анимации (GSAP + ScrollTrigger), «живые» эффекты, параллакс, shine-кнопки, усиленные тени и градиенты. Исходные файлы не изменяются.

## Как посмотреть премиум-версию

**Не открывайте файл двойным кликом** (file://) — скрипты-модули и стили могут не подгрузиться.

Запустите локальный сервер из папки проекта и откройте в браузере **index-premium.html**:

```bash
# Из папки portfolio-k8s:
npx --yes serve -l 3000
# или
python3 -m http.server 3000
```

Затем откройте: **http://localhost:3000/index-premium.html**

Сделайте жёсткое обновление (Ctrl+Shift+R или Cmd+Shift+R), чтобы сбросить кэш.

**Как понять, что премиум включён:** в правом верхнем углу должен быть фиолетовый бейдж **«Premium»**. Если его нет — в адресной строке должно быть именно `index-premium.html`, а не `index.html`, и обновите страницу с очисткой кэша.

## Как включить на продакшене

Открывайте **index-premium.html** по нужному URL или настройте сервер на отдачу этой страницы.

Подключаются:
- **styles-premium.css** — после styles.css
- **app-premium.js** — после app.js
- **GSAP + ScrollTrigger** (CDN) и **gsap-premium.js** — анимации при скролле: появление блоков `.reveal`, карточек, групп навыков, списка «Сильные стороны» (stagger).

## Что добавлено

1. **Hero**
   - Подзаголовок контрастнее и чуть крупнее (17.5px).
   - Карточки «Фокус / Город / Контакты» — больше вертикального воздуха (padding, line-height).

2. **CTA «Связаться»**
   - При hover — лёгкий сдвиг текста вправо и появление стрелки →.

3. **Фото (аватар)**
   - Мягкая тень без видимой рамки, чистый crop (clip-path).
   - При hover — подъём на 8px и усиление тени.

4. **Навигация**
   - Плавно двигающийся подчёркивающий индикатор под активным пунктом (scroll spy).
   - При скролле шапка чуть сжимается (класс `header--scrolled`).

5. **Карточки и фон**
   - Лёгкий «живой» градиент при hover (drift).
   - Hero-фон — еле заметный drift.

6. **Список «Сильные стороны»**
   - Класс `reveal-stagger`: пункты списка появляются по очереди при скролле.

7. **Секции Обо мне / Навыки / Проекты / Опыт / Контакты**
   - Остаются с существующим `.reveal` из основного app.js.

8. **ULTRA (в styles-premium.css + gsap-premium.js)**
   - Hero: градиентная вуаль, усиленное свечение подчёркивания имени, pills с лёгким glow при hover.
   - Primary CTA: shine-полоска при hover, лёгкий подъём.
   - Карточки: подъём на 3px и глубже тень при hover.
   - Разделители секций: ярче градиент и свечение.
   - Scroll progress: градиент и glow.
   - Кнопка «Наверх»: подъём + scale при hover.
   - Contact card: свечение и лёгкий подъём при hover.
   - Footer: градиентная верхняя граница.
   - Hero background: лёгкий параллакс при скролле (GSAP).

## Файлы

| Файл | Назначение |
|------|------------|
| **index-premium.html** | Страница с подключёнными premium-стилями и скриптом, разметка для CTA и списка. |
| **styles-premium.css** | Все премиум-стили (переопределения и новые правила). |
| **app-premium.js** | Scroll spy для навигации, уменьшение шапки при скролле, observer для `.reveal-stagger`. |
| **gsap-premium.js** | GSAP + ScrollTrigger: анимация появления при скролле для `.reveal`, `.cards .card`, `.skillgrid .skillgroup`, списка `.reveal-stagger` (stagger). Подключаются скрипты GSAP и ScrollTrigger с CDN, затем gsap-premium.js. |

Оригинальные **index.html**, **styles.css** и **app.js** не меняются.
